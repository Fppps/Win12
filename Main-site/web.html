Build 060

Official 1H06 Release

Focus AreaOfficial Release, Taskbar Placement, 3D Visuals, Login Storage
Status Released
Credit@Fppps

Windows 12 1H06 is here. This release restores the updated 3D look, expands taskbar controls, and seamlessly adapts OS colors to your wallpaper automatically.

Additions

  • Windows 11 K2 Features: Implements real-world K2 features, including variable taskbar placement.
  • 3D Visual Return: Reimplemented the 1H04 3D aesthetic with an updated, modern style.
  • OS Color Adapter: The interface now automatically tints based on your wallpaper—no manual selection needed.
  • Local Login Storage: Added a local credential storage system to retain session data.

Login Recovery Guide

  1. Press F12 to open Developer Tools.
  2. Navigate to the Application tab.
  3. Expand Local Storage in the sidebar.
  4. Locate the key win12_settings.
  5. Copy your saved password from the stored JSON value.

Build 040

Official 1H05 Release

Focus AreaOfficial Release, UI Polish, Core Stability
Status Released

Windows 12 1H05 is Official. This release solidifies the rebuild branch, bringing a stabilized split taskbar, a functional quick settings menu, and comprehensive system UI refinement to the public.

Release Highlights

  • Stable Taskbar Dynamics: Finalized the split floating taskbar behaviors and transition animations.
  • Quick Settings Deployment: Fully integrated volume, brightness, and toggle controls directly into the system tray.
  • Search Stability: Search bar logic is fully connected and performance-optimized for the desktop.

Build 035

Release Preview

Build 035 brought the architecture closer to release stability. The subsequent Build 040 served as the official public release.

Changelog Notes

  • Taskbar Modes: Introduced toggles between classic flat and the new split floating taskbar.
  • Quick Settings: Added the taskbar flyout with Night Light controls and volume/brightness sliders.
  • Search Bar Return: Re-integrated desktop search with functional UI elements.
