1H05 Rebuild Documentation

This document contains the official patch notes and structural documentation for the Webwindows 12 "Rebuild" updates. Features are listed chronologically below.

1H05: Build 030

"The Intentionality Update"

Release Preview
Release Date Today - Evening
Focus Area Interactive UI, Storage Handling, Widget Fixes
Status Stable Build
Roadmap Path to Official Release (Build 040)

Roadmap Update: Build 030 brings us significantly closer to release. Build 040 will serve as the official public release version and is currently in active development.

Changelog Notes

  • Calendar Interactivity: Added the calendar flyout with interactive date selection and agenda features (functionality requires user exploration).
  • Quick Settings Return: Reintroduced the Quick Menu functionality, heavily redesigned from original concepts and ideas.
  • Settings Application: Activated the Settings app with working interactive elements, sliders, and color pickers.
  • Smart Taskbar Toggle: Shifted the taskbar hiding logic to the taskbar itself instead of the Start menu. Clicking the taskbar area makes it pop up; double-clicking dismisses it back into hiding.

1H05: Build 025

"The Glass & Edges Update"

Beta Channel
Release Date Today - Afternoon
Focus Area Visual Overhaul, CSS Radii, UI Standardization
Status Dependent (Available if downloaded)

Changelog Notes

  • Unified Settings Engine: The settings loop was unified, immediately mirroring your Mica color and Opacity choices across the entire OS.
  • Global Radius Redesign: Implemented a centralized --os-radius CSS variable set to 6px, replacing overly rounded components with a sharper, intentional look.
  • Theme-Aware Outlines: Sliders are now the exclusive elements featuring borders, with outlines dynamically mixing the system accent color with white for a frosted effect.
